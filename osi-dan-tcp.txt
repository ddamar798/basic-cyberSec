üî∑ 1. Apa Itu Model OSI?
OSI = Open Systems Interconnection
Adalah model konseptual yang membagi proses komunikasi jaringan menjadi 7 lapisan (layer).

| No  | Layer        | Fungsi                                 | Contoh               |
| --- | ------------ | -------------------------------------- | -------------------- |
| 7Ô∏è‚É£ | Application  | Interaksi dengan user                  | Browser, FTP, Telnet |
| 6Ô∏è‚É£ | Presentation | Format data (enkripsi, encoding)       | SSL, TLS             |
| 5Ô∏è‚É£ | Session      | Mulai dan kelola sesi                  | Session token        |
| 4Ô∏è‚É£ | Transport    | Kirim data dengan betul (TCP/UDP)      | TCP, UDP             |
| 3Ô∏è‚É£ | Network      | Rute antar jaringan                    | IP, Router           |
| 2Ô∏è‚É£ | Data Link    | Komunikasi antar perangkat di jaringan | MAC Address          |
| 1Ô∏è‚É£ | Physical     | Kabel, sinyal                          | Ethernet, Wi-Fi      |

üî∑ 2. Model TCP/IP
-----------------------------------------------
Model TCP/IP lebih sederhana dan dipakai di dunia nyata saat ini. Hanya terdiri dari 4 layer:
| OSI Layer | TCP/IP Layer   | Contoh         |
| --------- | -------------- | -------------- |
| 5‚Äì7       | Application    | HTTP, DNS, FTP |
| 4         | Transport      | TCP, UDP       |
| 3         | Internet       | IP             |
| 1‚Äì2       | Network Access | Ethernet, MAC  |

üìå Jadi: OSI = teori | TCP/IP = praktik

üîÑ 3. Bagaimana Data Bergerak di Jaringan?

Misalnya kamu buka YouTube:

1. Kamu klik link video (Application)
2. Data dikodekan (Presentation)
3. Dibuat sesi (Session)
4. Dipecah jadi paket (Transport)
5. Dikasih alamat tujuan (Network)
6. Disiapkan untuk dikirim lewat kabel (Data Link)
7. Dikirim via Wi-Fi/kabel (Physical)

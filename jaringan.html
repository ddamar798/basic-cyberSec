1. Apa itu jaringan : Jaringan adalah sekumpulan perangkat (komputer, router, printer) yang saling terhubung dan bisa saling bertukar Data. Jenis-jenis Jaringan : - LAN (Local area Network) - WAN (Wide Area Network) -> Jaringan Internet Global. ===================================================================================== ===================================================================================== KOMPONEN PENTING JARINGAN | --------------- | -------------------------------------------------------------- | | **IP Address** | Alamat unik setiap perangkat | | **MAC Address** | Alamat fisik perangkat (NIC) | | **Router** | Penghubung antara jaringan (LAN ↔ Internet) | | **DNS** | Menerjemahkan nama domain → IP | | **Gateway** | Jalur keluar dari jaringan lokal | | **Subnet Mask** | Menentukan bagian network dan host dalam IP | | **Ports** | Pintu komunikasi untuk layanan (HTTP: 80, HTTPS: 443, SSH: 22) | ============================================================ ============================================================ P R A K T K -------------------------------------------------------------- 1> Tulis hasil dari perintah terminal! 1.
